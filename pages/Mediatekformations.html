MediaTekDocuments

Projet de gestion de documents pour une médiathèque. Gestion des commandes de livres, suivi des exemplaires, et plus.

Langages et technologies utilisés

C#, .NET, API REST, MySQL, Postman

Contexte et demande

Ce projet a été réalisé dans le cadre d’un atelier de professionnalisation avec le CNED. L’objectif est de développer une application de gestion de documents pour une médiathèque : livres, DVD, revues. J’ai implémenté notamment la gestion des commandes, le suivi des exemplaires, ainsi que diverses règles de gestion côté métier.
